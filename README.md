## 🚀 **GETTING STARTED**

```bash
# Clone the repository
git clone https://github.com/Infinite-88/emergent--nova.git
cd emergent--nova

# Install dependencies
npm install

# Initialize the system
npm run init

# Start the AI Empire
npm start
```
